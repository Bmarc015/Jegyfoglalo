# Jegyfoglaló

- Ez az oldal futball mérkőzésekre való jegyfoglalást/vásárlást biztosít.
<br>
- Az oldalt lehet adminként kezelni, mérkőzéseket rögzíteni, módosítani. Vásárlóként az alapvető funkciók elérhetőek.
<br>
- Az oldalon megtalálhatóak a jövőben megrendezésre kerülő mérkőzések, illetve a jegyek foglalásához segítségül egy interaktív helyfoglaló térkép az adott stadionról, ahol a mérkőzést játszák.
<br>
- A kiválasztás után a jegyeket a vásárló a kosárba tudja helyezni, ahol egy összesítés után a végösszeget, illetve az adatait láthatja
